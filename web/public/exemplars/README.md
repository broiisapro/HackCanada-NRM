# Spectral exemplar images

Place these five reference images here so the upload analyzer can use few-shot classification:

- `continousspectrum.png` — continuous spectrum (no gas)
- `hydrogen.png` — Hydrogen
- `sodium.png` — Sodium
- `calcium.png` — Calcium
- `mercury.png` — Mercury

Copy them from `pyros-vision/` (same filenames). If any are missing, the app falls back to single-image analysis without exemplars.
